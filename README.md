# Fame by Sheeraz - Website Clone

A pixel-perfect clone of [famebysheeraz.com](https://www.famebysheeraz.com/) built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Font:** Inter (via next/font)
- **Deployment:** Vercel

## Features

- Dark luxury theme with gold accents
- Responsive design (mobile, tablet, desktop)
- Scroll-based fade-in animations (IntersectionObserver)
- Auto-scrolling media logo marquee
- Sticky header with backdrop blur on scroll
- Case study grid with hover overlay effects
- Mobile hamburger menu
- Optimized images via Next.js Image component

## Sections

1. **Header** - Sticky navigation with CTA button
2. **Hero** - Full-screen hero with celebrity thumbnails
3. **As Seen On** - Infinite scrolling media logos marquee
4. **Services** - Service cards with icons (Celebrities, PR, Crypto, AI, Networking)
5. **Specialized Services** - Entrepreneur, Web3, PR, and FAME 911 offerings
6. **Case Studies** - 16 celebrity case study cards with hover effects
7. **About** - Bio section for Sheeraz Hasan
8. **Contact** - WhatsApp, email, and booking CTA
9. **Footer** - Social links, navigation, and branding

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```
